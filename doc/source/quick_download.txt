- CoSMoMVPA Matlab / Octave code:

    + ``git`` users::

        git clone https://github.com/CoSMoMVPA/CoSMoMVPA.git
        make -C CoSMoMVPA install

- CoSMOMVPA documentation in HTML format (for off-line viewing) is available in `zip <http://cosmomvpa.org/_static/CoSMoMVPA_documentation_html.zip>`_ and `gzipped tar archive <http://cosmomvpa.org/_static/CoSMoMVPA_documentation_html.tar.gz>`_.
